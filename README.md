# Quote Generator
> Simple quote generator using [QuoteDB](http://www.quotedb.com/) as a reference.
>[Inspiration](https://github.com/sananth12/Quote-of-the-Day).

## Install
```
$ npm install quote_generator
```

## Usage

```
$ npm run generate
```
